# Accessibility in Browsers

Modern browsers include tools that make websites easier to read and use.

These tools can help when:
- text is too small
- pages are cluttered
- reading long articles is tiring
- a website is written in another language

Many of these features are already built into your browser.

---

## Zoom

Zoom allows you to make a webpage larger so text is easier to read.

You can find the zoom controls in the **browser menu (three dots)**.

---

## Reader Mode

Some browsers offer **Reader Mode**, which removes adverts and distractions from pages.

This leaves only the main text and images, making articles easier to read.

---

## Page Translation

Many browsers can translate websites automatically.

This can be helpful when reading foreign websites.

---

## Text-to-Speech Extensions

Some browser extensions can read website text aloud.

These can help people with:
- vision difficulties
- dyslexia
- long articles

---

:::tip Did you know?
Many accessibility tools built into browsers were originally designed for people with disabilities but are useful for everyone.
:::

---

## Key Takeaway

Browsers include built-in tools that can make websites easier to read and understand.
