# Screen Readers Explained

A **screen reader** is software that reads the content of a website aloud.

It allows people who cannot easily see the screen to use computers and the internet.

---

## Who Uses Screen Readers?

Screen readers are commonly used by people who:

- are blind or partially sighted
- have severe vision difficulties
- prefer listening instead of reading

---

## How Screen Readers Work

Screen readers analyse the structure of a webpage and read:

- headings
- links
- buttons
- text

This allows users to navigate websites without needing to see the screen.

---

## Examples of Screen Readers

Some common screen readers include:

- **Narrator** (Windows)
- **VoiceOver** (Apple devices)
- **NVDA** (popular free screen reader)

---

:::tip Did you know?
Well-designed websites make it easier for screen readers to understand and read content correctly.
:::

---

## Key Takeaway

Screen readers allow people with vision difficulties to navigate websites by listening to the content.
