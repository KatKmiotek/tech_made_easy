# Accessibility on Your Device

Most computers, tablets, and smartphones include accessibility tools.

These tools can help make technology easier to use.

Examples include:

- larger text
- screen readers
- speech typing
- magnification tools
- colour and contrast adjustments

---

## Accessibility on Windows

Windows includes several built-in tools such as:

- **Magnifier** – makes parts of the screen larger
- **Narrator** – reads screen content aloud
- **High Contrast Mode** – improves readability

These can be found in the **Accessibility settings**.

---

## Accessibility on Apple Devices

Apple devices also include many accessibility tools.

Examples include:

- **VoiceOver** – screen reader
- **Zoom** – enlarges screen content
- **Spoken Content** – reads selected text aloud

---

:::tip Did you know?
Many accessibility tools are already built into your device. You do not need to install anything to use them.
:::

---

## Key Takeaway

Your computer or phone already includes tools designed to make technology easier to use.
