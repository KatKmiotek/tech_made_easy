# Browser Extensions

## What Is a Browser Extension?

A **browser extension** is a small tool you can add to your browser to give it extra features.

Think of extensions like **apps for your browser**.

They can add helpful tools such as:

- Ad blocking
- Translation
- Password management
- Reading assistance

Extensions appear as small icons near the top of the browser.

---

## Where Do Extensions Come From?

Extensions are usually installed from official stores such as:

- Chrome Web Store
- Microsoft Edge Add-ons
- Firefox Add-ons

These stores allow you to browse and install extensions safely.

---

## Examples of Useful Extensions

### Ad Blockers

Ad blockers remove many adverts from websites and can make pages easier to read.

Example: **uBlock Origin**

---

### Translation Tools

Translation extensions allow you to translate websites written in other languages.

Example: **Google Translate**

---

### Password Managers

Password managers help you store and fill in passwords securely.

Example: **LastPass**

These are especially useful for managing many different accounts.

---

### Read Aloud Tools

Some extensions can read website text aloud.

This can make long articles easier to follow.

---

:::tip Did you know?
Extensions can make your browser more powerful, but installing too many can slow your browser down.
:::

---

## Safety Tips

Before installing an extension:

- Install extensions from trusted stores
- Read reviews if available
- Avoid installing too many extensions
- Remove extensions you no longer use

---

## Key Takeaway

Browser extensions are small tools that add extra features to your browser.
