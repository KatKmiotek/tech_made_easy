# Useful Browser Settings

Most browsers include helpful settings that make websites easier to read and manage.

You can usually find these options in the **browser menu**.

The menu button normally looks like **three dots (⋯)** or **three lines (☰)** in the top corner of the browser. For Chrome, Edge and Firefox.  
Safari browser settings can be found on top bar. Safari -> Settings.

---

## Zoom – Make Text Bigger or Smaller

If a website looks too small, you can zoom in to make everything larger.

### How to change zoom

1. Click the **browser menu** (three dots or three lines).
2. Look for **Zoom** in the menu.
3. Click **+** to make the page bigger.
4. Click **−** to make the page smaller.

The page will immediately adjust.

Zoom works on almost every website.

For Safari use top bar -> View -> Zoom In (Zoom Out)

---

## Resetting the Zoom

If a page becomes too large or too small, you can return it to normal.

1. Open the **browser menu**.
2. Find the **Zoom** setting.
3. Set it back to **100%**.

100% is the normal view.

---

## Other Useful Things in the Browser Menu

The browser menu also gives access to many other helpful features.

You may see options such as:

### Bookmarks or Favorites

This is where you can view and manage websites you have saved.

---

### History

The history list shows websites you have visited recently.  
It can help you find a page again if you forgot to bookmark it.

---

### Downloads

If you download files such as documents or pictures, you can usually find them in the **Downloads** section.

---

### Search engine

You can choose which search engine the browser uses when you type in the address bar.

How to change the default search engine:

- Chrome / Edge:
	1. Open the **browser menu** → **Settings**.
	2. Go to **Search engine** (or `Search` section).
	3. Set **Search engine used in the address bar** and choose from the list (Google, Bing, DuckDuckGo, etc.).

- Firefox:
	1. Open the **menu (≡)** → **Settings** → **Search**.
	2. Under **Default Search Engine** pick one from the dropdown.

- Safari (macOS):
	1. Open **Safari** → **Settings** → **Search**.
	2. Choose the **Search engine** (Google, DuckDuckGo, or Yahoo).

Tip: If you care about privacy, try DuckDuckGo or another privacy-focused search engine.

### Clear Browsing Data

Browsers also allow you to remove stored information such as:

- Browsing history
- Cookies
- Cached website files

This can sometimes help fix problems with websites.

## Try This – Adjust the Zoom

1. Open a website.
2. Click the **browser menu**.
3. Increase the **Zoom** level.
4. Notice how the page becomes easier to read.

You can change the zoom again whenever you like.

---

## Common Confusion

• Zoom only changes how the page looks on **your screen**.  
• It does not change the website for other people.  
• Most browser tools can be found in the **menu button**.

---

## Key Takeaway

The browser menu contains many useful tools such as **zoom, bookmarks, history, and downloads**.
